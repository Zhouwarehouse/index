### nodejs版本切换使用

| 命令                |             说明             |
| ------------------- | :--------------------------: |
| nvm ls              | 查看所有已经安装的Nodejs版本 |
| nvm install 12.18.0 |         安装指定版本         |
| nvm use 版本号      |   切换版本（这个是全局的）   |
